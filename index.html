DFS Algorithm

Interest 11

I developed a SHAP-Based Dynamic Feature Engineering Algorithm, the research for which was published at the International Conference on Convergent & Smart Systems.

This algorithm uses SHapley Additive exPlanations—a concept rooted in game theory that fairly distributes contributions to predictions, assigning feature importance with precision. It dynamically selects the most relevant features for individual models in ensemble classifiers and generates synthetic features to mitigate class imbalance.

This approach improves both classification accuracy and interpretability (xAI). Built using Python and scikit-learn, it is particularly suited for applications in cybersecurity, healthcare and finance, where accurate and explainable predictions are critical.
